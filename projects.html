Projects

EasySave

I worked on a C# project with a group, where I was the primary contributor. We developed a WPF interface that allows users to automatically backup files or folders to another location. The program was designed to be user-friendly, with a simple and intuitive interface that allows users to easily configure backups and schedule them as needed.

Tétris

During my first year of DUT, I participated in a group project where we developed a Tetris game using Java. I was responsible for implementing the functional aspects of the application, as well as a significant portion of the JavaFX-based user interface. It was a challenging and rewarding experience, and I learned a lot about teamwork, software development, and problem-solving along the way.

Workout

In my second year of DUT, I worked with a partner on a Swift project that tracked the distance we traveled and time taken using a stopwatch. We built an iOS app that allowed us to enter start and end points, and displayed distance and time. The app also had features to save and share previous trips. We both contributed equally to the project, and it was a great learning experience in Swift programming and iOS app development.

Multiplication table

I have completed two PHP projects that showcase everything I have learned during my time in DUT. The first project is a classic multiplication tables website, while the second is an online store for fruits and vegetables. Both projects allowed me to apply my PHP skills and put my knowledge into practice. I am proud of what I have accomplished and look forward to continuing my development journey.

Menu updater

I developed a Python program called "Menu Updater" that updates the elements of the Beyond menu, the new project of Cegedim. The program sorts a JSON file containing all the menu items and sends the updates to the desired environment. I am proud of this program because it has proven to be extremely useful to my colleagues. However, I cannot share the code because it was developed for the company.
